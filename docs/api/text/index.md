# Text

### declension()
Склонение слова в зависимости от количества

**Examples:**

```ts
declension(1, ['слово', 'слова', 'слов']) // 'слово'
```

```ts
declension(2, ['слово', 'слова', 'слов']) // 'слова'
```

```ts
declension(5, ['слово', 'слова', 'слов']) // 'слов'
```
